My Advent of Code 2025 Solutions

My solutions for Advent of Code 2025, showcasing a modern Angular frontend with a Rust solver compiled to WebAssembly. This allows for high-performance code execution while leveraging Angular for a rich user interface. All computations are offloaded to a background Web Worker to ensure the UI remains responsive. You can find the source code on GitHub.

Deployed to both GitHub Pages and my self-managed Kubernetes cluster. You can also run the rust project locally without the Angular Frontent. See the README in the repository for instructions. Tests and benchmarks are included to ensure correctness and measure performance of the solutions.

Advent of Code is an annual programming puzzle series with daily challenges throughout December. To get the own personal puzzle inputs you need to sign up at adventofcode.com, there you can also find more information about the event
