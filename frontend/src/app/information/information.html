Advent of Code 2025

My solutions for Advent of Code 2025, showcasing a modern Angular frontend with a Rust-based solver compiled to WebAssembly. You can find the source code on GitHub.

Deployed to both GitHub Pages and my self-managed Kubernetes cluster. You can also run the rust project locally without the Angular Frontent. See the README in the repository for instructions.

Advent of Code is an annual programming puzzle series with daily challenges throughout December. To get the puzzle inputs you need to sign up at adventofcode.com, there you can also find more information about the event.
